HTML5


What does HTML Stand for?

HyperText Markup Language.

What is HTML used for?

Creating the structure of a webpage, with building blocks known as Elements.

What is an Element?

An element represents an individual component of an HTML document, like an image, text, or button. These elements are represented with tags.

What is an Attribute?

An Attriute describes more information about an element. Attributes are set in the opening tag of every element.

CSS3


What does CSS Stand for?

Cascading Style Sheet.

What is CSS used for?

Describing the style of HTML Elements on a webpage.

What is structure of a CSS Declaration?

selector { property: value }

What are three different ways to style a webpage with CSS

Inline, Internal, and External

What is the difference between a CLASS and an ID

A Class is a CSS Declaration that can be applied to many different elements. A an ID is a CSS Declaration that only applies to one unique element on a page.
